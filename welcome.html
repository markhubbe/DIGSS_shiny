Welcome to DIGSS

This shiny version of DIGSS was created to facilitate the user interface. More details on DIGSS can be found in Pestle et al. XXXX (link here soon).

DIGSS can be run as a package directly in R. You can install it following:

devtools::install_github("markhubbe/DIGSS")

The package in R will give you more flexibility when running multiple simulations, however the interface is much more cumbersome

You can check out all the code behind DIGSS on github:

For the R package: github.com/markhubbe/DIGSS

Comments and suggestions are always welcome. Send them to Mark Hubbe

To run DIGSS, simply input the parameters for the simulation and press the Run DIGSS button. The Survey loop will allow you to run multiple sequential simulations, at once. Soon it will be able to save results.
