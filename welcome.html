Welcome to DIGSS

This shiny version of DIGSS was created to facilitate the user interface. More details on DIGSS can be found in Pestle et al. XXXX (link here soon).

Help and walkthroughs are also available in the DIGSS vignette, accessible inside R or here.

DIGSS can be run as a package directly in R. You can install it following:

devtools::install_github("markhubbe/DIGSS")

The package in R gives more flexibility when running multiple simulations and further analyzing the results, however the interface is much more cumbersome

You can check out all the code behind DIGSS on github:

For the R package: github.com/markhubbe/DIGSS

For the Shiny version: github.com/markhubbe/DIGSS_shiny

Comments, suggestions and bug reports are always welcome. Send them to Mark Hubbe

To run DIGSS, simply input the parameters for the simulation and press the Run DIGSS button. The Survey Loop allows you to run multiple sequential simulations.

Soon DIGSS will be able to save results!

Warning for multiple users or intense simulations

A note for users interested in using this version of DIGSS with multiple students simultaneously or for those with extremelly large simulation goals:

The shiny.apps server has a limited ammount of memmory allocated to DIGSS, which means that multiple simultaneous users will cause the page to freeze.

If you are planning on sharing this with multiple users (e.g., in a classroom) or have very ambitious simulation goals, you can download the latest version of DIGSS_shiny as a zip file to load it in RStudio (unzip folder, open app.R in RStudio, and run app from there), or run it directly from github:

shiny::runGitHub('DIGSS_shiny', 'markhubbe')
